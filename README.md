# Sunrinlife Template
Template for SunrinLife <br>
SunrinLife is Sunrin Internet High School's intranet made by their students

## 🛠️ Installation

```bash
$ npm install
```

<br/>

## 🚀 Running the app

```bash
# development
$ npm run dev

# production mode
$ npm run start
```

<br/>

## 🔒 Environment configuration

```
NODE_ENV

PORT

DB_NAME
DB_PORT
DB_HOST
DB_USERNAME
DB_PASSWORD

NEIS_API_KEY
```
